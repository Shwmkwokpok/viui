# VIUI

#### Metro UI style Toolkit build on Vue.js 2.X


### INSTALL
> npm install viui --save

### DOCS
[中文文档](https://huayan.site/ViUI)

### Quick Start
```javascript
import Vue from 'vue'
import Viui from 'viui'

Vue.use(Viui)
```

### Browswer Support
Modern browsers and Internet Explorer 9+.
